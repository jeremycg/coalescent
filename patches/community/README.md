# Community patches

Patches contributed by Coalescent users — bigger, fully-produced patches that
show the modules in a real musical context, unlike the minimal smoke-test demos
in `patches/`. They are **not** regenerated by the `tools/` scripts; they are
kept exactly as contributed.

These patches use modules from other (free, VCV Library) plugins. Rack will
list anything you're missing when you open the patch — subscribe/install and
reopen.

**No sound on first load?** Shared patches carry the *contributor's* audio
device (these expect an "AudioBox USB 96"). Click the AUDIO module and select
your own driver/device — that's all it is.

## fractalgee — generative ambient systems

Contributed by **fractalgee (Georg Carlson)** — thank you! Originally shared in
[issue #2](https://github.com/jeremycg/coalescent/issues/2). Both are
self-playing generative systems: a clock plus random voltage sources play the
Coalescent oscillators through filters, a full mix bus, and reverb — patch it,
press play, and let it evolve.

| File | Original name | Coalescent modules |
| --- | --- | --- |
| `fractalgee_coalesce_fun.vcv` | FG_R2_568_07_Coalesce_Fun | Axon, Soma, Haptik |
| `fractalgee_gendyn_fun.vcv` | FG_R2_569_02_CoalesceGendyn_Fun | GENDYN |

Additional plugins used (all free, in the VCV Library): alefsbits, Bogaudio,
Count Modula, Grande, Impromptu Modular, MindMeld, Squinky Labs, Stoermelder
PackOne, Surge XT, Valley, Venom, Zoxnoxious.
